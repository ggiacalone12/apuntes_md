# Que es programar ?
Programar es la acción de crear un **programa**
# Que es un programa ? 
Un programa son muchos **pasos** que tienen un resultado esperado 
- ( Resultado como por ejemplo: la calculadora sumó 2*2 y el *resultado* fue 4. )
- A Muchos pasos juntos se los llama: *Algoritmo*
- Un programa es un conjunto de algoritmos que se ordenan de una manera para alcanzar el resultado esperado.
  - Ejemplos de algoritmo:
    - _Lavar los platos:_
      - 1° Agarrar esponja
      - 2° Prender canilla,calienta
      - 3° Mojar esponja
      - 4° Enjabonar esponja
      - 5° Agarrar utensillo_de_cocina
      - 6° Mientras que el *plato esté sucio* : Frotar esponja y utensillo_de_cocina
  - El algoritmo anterior en *pseudocódigo* (pseudocodigo se le dice a estas palabras que usamos recien pero que utilizan formás de mencionar **lógica y sintaxis*.)
